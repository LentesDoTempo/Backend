# Backend - Documentacao Tecnica

Este diretorio centraliza a documentacao tecnica do backend.

## Indice

1. api-routes.md
   - Catalogo completo de endpoints.
   - Regras de autenticacao por rota.

2. backend-architecture.md
   - Estrutura de pastas, responsabilidades e fluxo de request.

3. setup-env.md
   - Variaveis de ambiente e execucao local com/sem Docker.

4. auth-and-reset-flow.md
   - Fluxo de autenticacao e recuperacao de senha.

## Escopo

- Stack: Node.js + Express + Sequelize.
- Banco: MySQL.
- Autenticacao: JWT.
- Upload/analise de imagens: Multer + Sharp + Gemini.
